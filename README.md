### 👨‍💻 About Me

I design and ship **enterprise-grade software** — the kind that organizations actually rely on in production.

My core work is building scalable APIs and microservices in **.NET Core with Clean Architecture**, paired with modern frontends in **React, Next.js, and Angular**. I've led development teams end-to-end under **Scrum methodology** — owning sprint planning, task distribution across developers, retrospectives, and full delivery cycles.

I've **reverse-engineered legacy codebases** to extract business logic, mapped out architectural failures, and rebuilt systems that are decoupled, secure, and maintainable. I don't patch — I redesign.

What I've shipped: an **airport billing platform with AI-based aircraft recognition** that automates invoicing, a **real estate CRM** with AI chatbot and omnichannel messaging, **government institutional dashboards**, cloud backup validators, and asset management systems — all with **AWS + Docker + Kubernetes + CI/CD** in the stack.

> I don't write code for demos. Everything I ship is in production, used by real organizations.

### ✈️ Flagship Project

<br/>

<table width="85%">

<td align="left" style="background:#0D2137;border-radius:12px;padding:20px">

<h3>🛫 AirportBill — Airport Billing Platform</h3>

Enterprise billing system for airports featuring an **AI-based aircraft recognition module** that automates invoice generation from real flight data. No manual input. The system identifies the aircraft, pulls the relevant billing rules, and issues the invoice automatically.

<br/>

| Layer | Stack |
|:---|:---|
| **Backend** | `.NET Core` · `Clean Architecture` · `C#` · `RESTful API` · `SQL Server` |
| **Frontend** | `React` · `TypeScript` · `Responsive Design` |
| **AI Module** | `Computer Vision` · `Aircraft Recognition` · `Automated Invoicing` |
| **Infra** | `Docker` · `CI/CD Pipeline` |

<br/>

> *Designed from the ground up with Clean Architecture — decoupled layers, domain-driven, testable.*
